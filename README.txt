The isocore style is a configuration for the OSM2World 3D renderer. It is used for generating the orthographic slippy map available on http://maps.osm2world.org

Textures were contributed by members of the OpenStreetMap community and released into the Public Domain.
Visit http://wiki.openstreetmap.org/wiki/Texture_Library to find out about the contributors and discover more texture images.

The configuration file has been released into the Public Domain (see http://creativecommons.org/publicdomain/zero/1.0/).
